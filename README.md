### Exercício 1

- Dificuldade: Médio
- [Caminhos Únicos](https://leetcode.com/problems/unique-paths/description/)

Dado um grid m x n e um robô posicionado na posição grid[0][0], calcule o número de caminhos únicos que o robô pode percorrer até a posição final do grid.


### Exercício 2

- Dificuldade: Difícil
- [Agendador de tarefas - Máximo Lucro](https://leetcode.com/problems/maximum-profit-in-job-scheduling/description/)

Dados um array de tasks à serem executadas e seu respectivo lucro atrelado, retorne o máximo lucro possível sem sobreposição das tarefas..
